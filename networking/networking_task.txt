2.TCP vs UDP : In This we saw that TCP is reliable, ordered and Slow and broadcasting not possible and its send the receipt of every data packet while UDP is not reliable, not in ordered way but fast, broadcasting is possible and did not send the receipt of data packet

6.TCP/IP and OSI Model 
In TCP/IP four layers are there like
Application Layer : What to send (Https, SSH, FTP)
Transport: How to send (TCP vs UDP)
Network: Where to Send the IP Address
Data Link: Ethernet, Wifi etc 

OSI Model: APSTNDP
Application
Presentation
Session
Transport
Network Access
Data Link
Physical Layer



Note: If a destination reach by Ip address and ping is also working but website is not open so the issue is DNS (Domain Name Server) to Check The Zone Editor of the domain to identify which record is failing
